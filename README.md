### This project is to perform quantitative analysis on Time Series Natural Gas Prices Dataset

## Data Analysis and Modelling
1. Analyzed 18 months time-series Natural gas prices dataset, to estimate the purchase price of gas at any
   date in the past and extrapolate it for one year into the future.
2. Constructed a function to take date as an input and return a price estimate. Computed the price estimate using
   a Linear Regression Model.
