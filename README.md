### This project is to perform quantitative analysis on Time Series Natural Gas Prices Dataset

## Data Analysis and Modelling (Natural_Gas_DataAnalysis.ipynb)
1. Analyzed 18 months time-series Natural gas prices dataset, to estimate the purchase price of gas at any
   date in the past and extrapolate it for one year into the future.
2. Constructed a function to take date as an input and return a price estimate. Computed the price estimate using
   a Linear Regression Model.

## Pricing Model Function 
1. Implemented a price_model function that calculates the profit or loss obtained by undertaking trades on given dates      for a contract involving the buying, storing, and selling of natural gas the storage cost of the gas, the injection/withdrawal. The value of the contract is the profit or loss obtained by undertaking the trades on given dates.
